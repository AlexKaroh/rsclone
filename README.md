# RS Clone

## Overview
Frontend part for the final RS School project


## Installation

```bash
$ npm install
```


### Running the app

```bash
# development
$ npm start

# build
$ npm run build
```

## Technical Stack
### Front-End
1. React
1. React-routing
1. MobX
1. i18next
1. TypeScript
1. ESlint, Prettier
1. Axios
